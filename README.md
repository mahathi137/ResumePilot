# ResumePilot 🚀

Build Smarter Resumes. Get Career Ready.

ResumePilot is a modern ATS-aware resume builder that helps students and job seekers create professional, recruiter-friendly resumes with real-time preview, resume analysis, and PDF export capabilities.

### Live Demo

🔗 https://tryresumepilot.netlify.app/

---

## Features

* Professional Resume Templates
* Real-Time Resume Preview
* ATS Score Analysis
* Readability Analysis
* Resume Strength Dashboard
* Dynamic Education, Experience, Projects, Skills & Certifications
* Custom Sections
* Multiple Resume Versions
* Professional Summary Generator
* Dark Mode & Light Mode
* Theme Customization
* PDF Export with Clickable Links
* Print-Friendly Resume Layout
* Fully Responsive Design

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* html2canvas
* jsPDF
* SortableJS

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/resumepilot.git
```

2. Open the project folder

```bash
cd resumepilot
```

3. Run `index.html` using Live Server or open it directly in your browser.

## Why ResumePilot?

ResumePilot is more than a resume builder. It helps users improve their resumes through ATS scoring, readability checks, keyword analysis, and professional templates inspired by industry standards.
